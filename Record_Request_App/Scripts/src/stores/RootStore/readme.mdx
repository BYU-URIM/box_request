# Root Store

## Store

### This is the base store where you can access all other stores.

---

First, this store grabs the user. The store then uses the information of the user to determine what boxes, departments, etc. the user will see. All other stores are accessed through RootStore.

This store accesses the
- User
- Checkout Store
- User Store
- UI Store
